
# Queue Waiter

A minecraft proxy which can be be used to wait out 2b2t's queue without having to have your game open

![Logo](https://i.imgur.com/DzmCIQU.png)

## Installation

```bash
  npm Install
  node .
```
## Example Config

```json
{
  "username": "email@gmail.com",
  "password": "Password123",
  "auth": "microsoft",
  "host": "2b2t.org",
  "version": "1.12.2",
  "local": "localhost"
}

```
